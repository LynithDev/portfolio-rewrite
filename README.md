# My personal website
Made with ✨ NextJS and TailwindCSS ✨

## 🚀 Getting Started
If using npm:
```bash
npm install && npm run dev
```

If using yarn:
```bash
yarn && yarn dev
```

If using pnpm:
```bash
pnpm install && pnpm run dev
```
